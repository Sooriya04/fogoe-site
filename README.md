# Fogoe

A minimal CLI for initializing Node.js backend projects with zero friction.

## Quick Start

```bash
npx fogoe
```

No installation required.

## Features

- **Frameworks:** Express, Fastify
- **Module Systems:** CommonJS, ES Modules
- **Databases:** MongoDB, Prisma, MySQL, PostgreSQL
- **Security:** bcrypt, argon2, CORS, dotenv
- **Architecture:** Minimal or MVC

## What It Does

Fogoe generates a working backend with:
- Server setup
- One route
- Database connection template
- Environment configuration

Reach a running state in seconds.

## Philosophy

Fogoe scaffolds projects, not full systems. No business logic, no schemas, no lock-in. You build the rest.
